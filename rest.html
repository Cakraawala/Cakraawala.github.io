Project REST API

Powerful REST API for accessing and managing data with high flexibility and optimal performance.

Technologies: PHP Laravel, MySQL

Documentation

    {
        "endpoint": "/api/",
        "method": "GET",
        "headers": {
            "Authorization": "API_KEY"
        }
    }

Endpoint: /api/

Method: GET

Headers: Authorization: API_KEY

Features

  • Powerful: Supports high flexibility for accessing and managing data.
  • Optimal Performance: Provides fast data access and processing.
  • Security: Requires API_KEY for authorized access.

Usage

        {
            "name" => "Angger Cakra"
            "username" => "Cakraawala"
            "email" => "Anggercakrawala@gmail.com"
            "api_keys" => [
              "key" => "HZ5PB2a2OzuEsAGtayPlynsl6YZ8X4"
              "start_time" => 1690617325
              "exp_time" => 1693209325
              "limit" => 100
            ]
        }
  1. Get 'key' from create account and get the api key.
  2. Use 'key' to access data through endpoint /api/data?apikey=yourapikey.
  3. Add request for a detail
    • Request ID to fetch 1 data based on the requested ID. 
      http://127.0.0.1:8000/api/mahasiswa?apikey=yourapikey&id=4
      http://127.0.0.1:8000/api/users/{ID}/cart
    • Request Name to fetch all data based on the request Name 
      http://127.0.0.1:8000/api/mahasiswa?apikey=yourapikey&name=Angger
    • Request Username to fetch all data based on the request Username 
      http://127.0.0.1:8000/api/users?apikey=yourapikey&username=Thomas Slebew
  4. There is 5 methode
    • Methode GET all data
    • Methode POST data. 
      http://127.0.0.1:8000/api/data?apikey=yourapikey
    • Methode PUT data for a Update.
    • Methode SHOW data for showing data with parameter ID.
    • Methode DELETE data 
      http://127.0.0.1:8000/api/data/{ID}?apikey=yourapikey

My Project REST API

REST API Users

Project REST API User menyediakan layanan pengelolaan data pengguna dengan menggunakan metode GET, SHOW, UPDATE, dan DELETE. Dengan menggunakan API ini, Anda dapat mengakses, menampilkan, memperbarui, dan menghapus data pengguna dengan mudah dan aman. Project ini memungkinkan integrasi yang mulus dengan aplikasi lain untuk mengelola data pengguna dengan efisien dan andal.

REST API Mahasiswa

API REST Mahasiswa: Akses, tampilkan, perbarui, dan hapus data mahasiswa dengan efisien dan mudah. Integrasikan dengan aplikasi lain untuk pengelolaan data yang lebih baik.

REST API Cart

API REST Cart: Kelola keranjang belanja dengan mudah. Tambahkan, tampilkan, perbarui, dan hapus item dengan cepat. Manfaatkan fitur API untuk pengalaman berbelanja yang lancar.
